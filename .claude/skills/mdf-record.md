# mdf-record

## 触发条件

由 Claude Code Stop Hook 自动触发。

## 执行步骤

1. **从对话提取摘要**：
   - 阅读本次对话/任务的内容
   - 提炼关键信息：任务目标、做了什么、为什么这样做、决策、结果

2. **写入变更记录**：
   - 确保 `docs/mdf/` 目录存在
   - 追加记录到 `docs/mdf/changes.md`，格式：
   ```
   ---
   ## {时间ISO格式}
   
   **任务**: {任务描述}
   
   ### 改动原因
   {原因}
   
   ### 决策
   {做了哪些决策}
   
   ### 结果
   {完成什么}
   ```

3. **更新计数**：
   - 读取 `docs/mdf/.count`，数值 +1，写回
   - 若文件不存在，则创建并写入 1

4. **检查触发条件**：
   - 读取 `docs/mdf/changes.md` 行数
   - 若 计数 >= 10 **或** 行数 >= 2000：
     - 自动调用 `/mdf-summarize` 执行汇总
     - 汇总完成后重置计数为 0
   - 否则结束
